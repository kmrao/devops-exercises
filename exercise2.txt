
Login into AWS and Create EC2 instance and perform the below steps
Step1:   Install  Java.
Step2:   Install Tomcat
Step3:   Install the Docker
Step4:   Download Jenkins war file and copy to tomcat
Step5:   Start the Tomcat and login to jenkins
Step6:   Create new Maven Project
Step7:   Build the project   https://github.com/kmrao/usermanagement.git by using pom.xml
Step8:   Build the Docker Image for the above project output with tag usermanagement:1.0
Step9:   Push the Docker Image to Docker Hub.
Step10:  Down load the Docker Image and start the server and  make sure that all the endpoints are working properly.
